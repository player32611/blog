## 什么是机器学习

对于有些问题，我们无法用确定的逻辑编程实现。如图像识别，语音识别，垃圾邮件分类。

机器学习的核心思想是模拟人的学习能力，从样本数据中学习，得到经验/模型，然后进行预测，这是一种数据驱动的方法。

## 基本概念

**样本** : 机器学习算法处理的数据

**特征向量** : 人工构造的用于描述一个样本的向量，如颜色、形状等

**预测函数** : 实现从样本的特征向量到预测值的映射

**目标函数** : 训练时使用，目的是确定模型的参数

**监督信号** : 人工为样本打上的标签（label），如图像的类别，是 26 个英文字母中的哪一个

## 机器学习算法的分类

### 有监督学习

有监督学习事先用带有标签的样本进行训练，然后用得到的模型预测。

有监督学习能根据样本学习得到一个映射函数 y = f(x)。

有监督学习常用来解决**分类问题**与**回归问题**

**分类问题** : 确定样本的类别，如人脸识别，字符识别，语音识别。类别标签是整数编号。

**回归问题** : 确定一个实数值，如根据一个人的学习、工作年限、行业等信息预测他/她的收入。

### 无监督学习

无监督学习则直接对数据进行预测，样本不带有人工标注的标签值。

无监督学习常用来解决**聚类问题**与**数据降维问题**

**聚类问题** : 将一批样本分成多个类，即多个不相交的子集，每个样本属于其中的一个子集。类别没有事先定义，而是由算法确定。

**数据降维问题** : 降维是指将高维数据映射到低维数据，如将二维数据映射到一维数据。

### 强化学习

强化学习根据当前的状态确定要执行的动作，以达到某一目标，抽象来说，是最大化累计回报。

## 一般流程

### 有监督学习

1. 先采集训练样本

2. 得到函数模型 F(x)（带有参数）

3. 把新的样本带到模型中，得到预测值
